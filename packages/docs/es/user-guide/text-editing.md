# Edición de texto

## Crear texto
Pulsa **T**, luego haz clic en el lienzo. Empieza a escribir inmediatamente.

## Edición inline
Doble clic en un nodo de texto para entrar en modo edición. Clic fuera para confirmar.

## Navegación del cursor
| Acción | Mac | Windows / Linux |
|--------|-----|-----------------|
| Izquierda/derecha | <kbd>←</kbd> / <kbd>→</kbd> | <kbd>←</kbd> / <kbd>→</kbd> |
| Arriba/abajo | <kbd>↑</kbd> / <kbd>↓</kbd> | <kbd>↑</kbd> / <kbd>↓</kbd> |
| Por palabra | <kbd>⌥</kbd><kbd>←</kbd> / <kbd>⌥</kbd><kbd>→</kbd> | <kbd>Ctrl</kbd> + <kbd>←</kbd> / <kbd>Ctrl</kbd> + <kbd>→</kbd> |
| Inicio/fin de línea | <kbd>⌘</kbd><kbd>←</kbd> / <kbd>⌘</kbd><kbd>→</kbd> | Inicio / Fin |

<kbd>Shift</kbd> extiende la selección.

## Formato enriquecido
| Acción | Mac | Windows / Linux |
|--------|-----|-----------------|
| Negrita | <kbd>⌘</kbd><kbd>B</kbd> | <kbd>Ctrl</kbd> + <kbd>B</kbd> |
| Cursiva | <kbd>⌘</kbd><kbd>I</kbd> | <kbd>Ctrl</kbd> + <kbd>I</kbd> |
| Subrayado | <kbd>⌘</kbd><kbd>U</kbd> | <kbd>Ctrl</kbd> + <kbd>U</kbd> |

## Selector de fuentes
Búsqueda, vista previa y scroll virtual. Fuentes del sistema en desktop (Tauri), Local Font Access API en el navegador.
